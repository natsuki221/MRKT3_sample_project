# MRKT3_Sample_Project

## 此為*MRTK*預先安裝完成的套件
### 版本訊息：
```
1. Unity 2021.3.28f1
2. MRTK
  * Ver3.0.0
  * Ver2.8.3
3. MixedRealityFeatureTool v1.0.2209.0 Preview
4. Google NatoSans TC
```

### 安裝方式
1. 將此壓縮檔下載至電腦
2. 解壓縮
3. 打開*Unity Hub*
4. 點選**開啟**
5. 選擇剛已解壓縮的資料夾
